# 컬럼 매핑 가이드

ROAS 분석기는 다양한 광고 플랫폼의 데이터를 자동으로 인식합니다.

## 📊 지원하는 데이터 형식

### 1. 메타(Facebook) 광고 관리자
```
일 → 날짜
지출 금액 (KRW) → 광고비
구매 전환값 → 매출
구매 → 전환수
링크 클릭 → 클릭수
노출 → 노출수
```

### 2. 구글 애즈
```
Date → 날짜
Cost → 광고비
Conversion value → 매출
Conversions → 전환수
Clicks → 클릭수
Impressions → 노출수
```

### 3. 네이버 광고
```
날짜 → 날짜
광고비 → 광고비
매출 → 매출
전환수 → 전환수
클릭수 → 클릭수
노출수 → 노출수
```

### 4. 카카오 광고
```
일자 → 날짜
지출 → 광고비
전환값 → 매출
전환 → 전환수
클릭 → 클릭수
노출 → 노출수
```

## 🔍 인식 가능한 컬럼명 전체 목록

### 날짜 (필수)
- `date`, `날짜`, `일자`, `Date`, `DATE`
- `일`, `보고 시작`, `리포팅 시작`, `Day`

### 광고비 (필수)
- `adSpend`, `ad_spend`, `광고비`, `광고비용`
- `cost`, `Cost`, `spend`, `Spend`
- `지출 금액 (KRW)`, `지출 금액`, `지출`
- `Amount spent`, `Amount Spent`

### 매출 (필수)
- `revenue`, `매출`, `수익`
- `Revenue`, `REVENUE`, `sales`, `Sales`
- `구매 전환값`, `전환값`
- `Purchase conversion value`, `Conversion value`

### 전환수 (필수)
- `conversions`, `전환수`, `전환`
- `Conversions`, `CONVERSIONS`, `conv`
- `구매`, `Purchase`, `Purchases`, `Orders`

### 클릭수 (필수)
- `clicks`, `클릭수`, `클릭`
- `Clicks`, `CLICKS`
- `링크 클릭`, `Link clicks`, `Link Clicks`
- `Outbound clicks`

### 노출수 (선택)
- `impressions`, `노출수`, `노출`
- `Impressions`, `IMPRESSIONS`, `impr`
- `Reach`, `도달`

## ✅ 사용 예시

### 메타 광고 데이터
```csv
일,캠페인 이름,광고 세트 이름,광고 이름,광고 게재,지출 금액 (KRW),노출,링크 클릭,구매,구매 전환값,장바구니에 담기,장바구니에 담기 전환값,보고 시작,보고 종료
2025-12-15,캠페인A,광고세트1,광고1,ON,100000,5000,500,45,450000,100,200000,2025-12-15,2025-12-15
```

**자동 인식:**
- `일` → 날짜
- `지출 금액 (KRW)` → 광고비
- `노출` → 노출수
- `링크 클릭` → 클릭수
- `구매` → 전환수
- `구매 전환값` → 매출

### 필수 컬럼만 있는 간단한 데이터
```csv
날짜,광고비,매출,전환수,클릭수
2025-12-15,100000,450000,45,500
```

## ⚠️ 주의사항

### 1. 필수 컬럼
다음 5개 컬럼은 **반드시** 있어야 합니다:
- 날짜
- 광고비
- 매출
- 전환수
- 클릭수

### 2. 노출수 (선택)
- 노출수가 없어도 분석 가능
- 있으면 CTR(클릭률) 계산 가능

### 3. 추가 컬럼
- 분석에 사용되지 않는 컬럼이 있어도 괜찮습니다
- 예: 캠페인 이름, 광고 세트 이름 등

### 4. 날짜 형식
지원하는 날짜 형식:
- `2025-12-15` (권장)
- `2025/12/15`
- `12/15/2025`
- 엑셀 날짜 (자동 변환)

## 🔧 컬럼명이 인식 안 될 때

컬럼명이 위 목록에 없다면:

### 방법 1: 엑셀에서 컬럼명 변경
```
Before: Cost (USD)
After:  cost
```

### 방법 2: 표준 컬럼명 사용
아래 컬럼명을 사용하면 항상 인식됩니다:
```csv
date,adSpend,revenue,conversions,clicks,impressions
```

### 방법 3: 한글 컬럼명 사용
```csv
날짜,광고비,매출,전환수,클릭수,노출수
```

## 📝 메타 광고 데이터 다운로드 가이드

1. 메타 광고 관리자 접속
2. 왼쪽 메뉴에서 "광고" 선택
3. 상단의 날짜 범위 선택
4. 필요한 컬럼 선택:
   - 일
   - 지출 금액 (KRW)
   - 노출
   - 링크 클릭
   - 구매
   - 구매 전환값
5. "내보내기" → "선택한 데이터를 Excel로 내보내기"

**완료!** 다운로드한 파일을 그대로 업로드하면 됩니다.
