# 데이터테이블 확대 기능 추가

## 작업 개요
계층별 광고 요약 테이블에 확대 버튼을 추가하여 전체 화면 모달로 테이블을 크게 볼 수 있는 기능 구현

## 작업 일자
2026-01-29

## 수정 파일
- `src/views/admin/dataTables/components/HierarchicalAdSummary.js`

## 구현 내용

### 1. 확대 버튼 추가
- 위치: 매체선택 버튼 왼쪽
- 아이콘: `MdZoomOutMap` (확대 아이콘)
- 스타일: 매체선택 버튼과 동일한 디자인 적용

### 2. 전체 화면 모달 구현
- Chakra UI의 Modal 컴포넌트 사용
- `size="full"` 속성으로 전체 화면 크기 설정
- 마진 20px 적용하여 화면 가장자리 여백 확보

### 3. 모달 헤더
- 타이틀: "계층별 광고 요약 - {현재 탭명}"
- 우측에 매체 필터 및 탭 버튼 배치
- 닫기 버튼(X) 포함

### 4. 테이블 렌더링 최적화
- `renderTable()` 함수로 테이블 렌더링 로직 추출
- 일반 화면과 모달에서 동일한 테이블 렌더링 함수 재사용
- 코드 중복 제거 및 유지보수성 향상

### 5. 모달 내 기능
- 매체 필터: 전체/Google/Naver/Meta 등 선택 가능
- 탭 전환: 캠페인/광고그룹/광고 탭 전환 가능
- 페이지네이션: 페이지 이동 기능 포함
- 비교 모드: 현재 기간/비교 기간/증감 데이터 모두 표시
- 스크롤: 세로 스크롤 지원 (`maxH='calc(100vh - 200px)'`)

### 6. 사용자 경험 개선
- ESC 키로 모달 닫기
- 모달 외부 클릭으로 닫기
- 닫기 버튼(X) 클릭으로 닫기
- 다크모드 지원

## 기술 스택
- React Hooks: `useState`, `useEffect`, `useMemo`, `useDisclosure`
- Chakra UI: Modal, IconButton, Menu 등
- React Icons: MdZoomOutMap 아이콘
